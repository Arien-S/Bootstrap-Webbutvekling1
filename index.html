Element


Ett HTML-element är en komponent i ett HTML-dokument som talar om för en webbläsare hur man strukturerar och tolkar en del av HTML-dokumentet


HTML


HTML, eller HyperText Markup Language, låter oss strukturera stycken och länkar med hjälp av element, taggar och attributes för att skapa hemsidor.


ID


Id-attributet anger ett unikt id för ett HTML-element. Värdet på id:t måste vara unikt i HTML-dokumentet. id:t används för att hitta en specifik style från css. Det används också av JavaScript för att komma åt och ändra elementet med det specifika ID:t.


Class


Class är ett attribut som används för att klassificera element. Till skillnad från id så kan det finnas flera element med samma klass på en sida. Man kan alltså föreställa sig att man grupperar element med varandra.


Meta


Ordet meta betyder "en underliggande definition eller beskrivning". En metadiskussion är en diskussion om en diskussion alltså så handlar det om sig själv.


Responsive


Responsiv webbdesign (RWD) är en betyder att man skapar dynamiska förändringar av utseendet på en webbplats, beroende på skärmstorleken och orienteringen på enheten som används för att visa den. Detta är extra viktigt så att mobiler,datorer och andra enheter ser hela hemsidan utan att vissa delar bryts av eller flyttas runt.


Http


HTTP står för Hypertext transfer protocol och är ett TCP-baserat protokoll som används för att hämta resurser som HTML-dokument. Det är grunden för allt datautbyte på webben och det är ett client-server protokoll, vilket innebär att requests sker av mottagaren, vanligtvis webbläsaren.


Https


HTTPS står för Hypertext transfer protocol secure och är den säkra versionen av HTTP, som är det primära protokollet som används för att skicka data mellan en webbläsare och en webbplats. HTTPS är krypterad för att öka säkerheten vid dataöverföring.


DNS


DNS står för Domain Name Service och är ett system för att förenkla adressering av datorer på IP-nätverk genom att kopplar ihop domännamn med IP-adresser. Tex Youtube.com tar dig till en specifik ip adress för servern


CDN


CDN står för content delivery network. Ett CDN möjliggör snabb överföring av tillgångar som behövs för att ladda innehåll på nätet inklusive HTML-sidor, javascript-filer, stilmallar, bilder och videor.


Compressed data


Datakomprimering är processen att modifiera/konvertera bit strukturen av data så att den förbrukar mindre utrymme på disk tex. Alltså att man minskar lagrings storleken på tex en fil. Det kan även kallas source coding eller bit-rate reduction.


Local Bootstrap


Bootstrap-webbplatser som använder lokal bootstrap använder en enda server, vilket kan sakta ner informationsöverföringen.Till skillnad från när man använder CDN.


jQuery


Är ett Javascript library som gör det lättare för en att andvända javascript på sin hemsida. Istället för att skriva en massa rader kod kan den förenkla vissa saker som kan kallas med en enda rad kod.jQuery förenklar också mycket av de komplicerade sakerna från JavaScript, som AJAX-calls och DOM-manipulation.


Flexbox


Är ett sätt att lättare fixa sin layout på sin hemsida utan att använda float eller positioning. Flexbox layouten är endimensionell och är väldigt användbar då man vill göra en responsiv webbsida, samt enklare kunna ändra varje del på hemsidan.


Grid


Likt flexbox så kan man använda grid för att lättare fixa sin layout på sin hemsida utan float eller positioning. Den stora skillnaden är dock att grid är tvådimensionellt medans flexbox är endimensionellt. Detta ger grid mer frihet i sätten man kan positionera sina items och containers.


Code Snippets


Snippet är en programmerings benämning för en liten del med återanvändbar källkod, maskinkod eller text. Code snippets är mallar som gör det enklare att ange upprepade kodmönster, såsom loopar eller villkorssatser.


WASM


WebAssembly är en open standard som tillåter exekvering av binär kod på webben. Denna standard tillåter utvecklare att ta prestanda för språk som C, C++ och Rust till webbutveckling området. Denna teknik används ofta för att utföra krävande operationer i webbläsaren.


Chromium


Chromium är ett gratis open source webbläsar projekt som utvecklats samt upprätthålls av Google. Mycket av chromiums källkod har använts för att göra andra kända webbläsare vi ofta ser eller använder tex Google Chrome, Microsoft Edge eller Opera.


SVG


SVG står för Scalable Vector Graphic och är ofta associerat med filer. En SVG fil är alltså en fil som till skillnad från tex en png fil inte använder unika pixlar för att skapa bilderna vi ser utan de använder istället vector data. Detta gör att dessa svg filer lätt kan skalas upp och ned utan att de förlorar sin bildkvalitet som ofta händer med png eller jpg filer. Därför är svg filer vanligt att använda inom web design där tex logos används på flera sidor i olika storlekar.


PNG24


PNG filer är ganska vanliga att använda då man tex vill lägga en bild på en annan bild då bakgrunden i png filer kan göras genomskinliga till skillnad från JPG filer. PNG 24 är ett slags filformat likt PNG 8 de de båda bibehåller skärpan i bilden väldigt bra. Skillnaden mellan PNG 24 och PNG 8 är att ena har 24 bit färgdjup medans den andra endast har 8 bit färgdjup. Detta gör även att PNG 24 bilder tar upp mer lagringsutrymme på datorn vilket kan behöva undvikas om man ska dela en bild till en stor
